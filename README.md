# 长三角地区土地利用变化及建设用地驱动性分析
![做蛋糕的奶龙](https://i1.hdslb.com/bfs/new_dyn/a37a7573d83993c36ab998d6be5d8ca2628576425.gif@264w_264h_1e_1c.webp)

## 📋 项目概述
本项目是华中科技大学2025～2026学年第一学期“GIS城市大数据”课程的结课论文成果，聚焦长三角地区（安徽、江苏、浙江、上海四省/市），基于2000-2020年多期遥感影像与地理空间数据，运用ArcGIS空间分析技术、Python数据分析工具及地理探测器模型，系统探究区域土地利用类型的时空演变特征，定量识别建设用地扩张的核心驱动因素，为区域土地资源合理规划与城市化可持续发展提供数据支撑。

## 🔧 核心技术栈
- 空间分析工具：ArcGIS Pro（叠加分析、缓冲区分析、栅格计算、要素裁剪、批处理等）
- 数据分析语言：Python（Pandas、NumPy、Matplotlib、Seaborn）
- 核心模型：土地利用转移矩阵、地理探测器模型（因子探测、交互探测）
- 数据处理：遥感影像裁剪/校正/配准、矢量数据投影统一、数据标准化与空间化表达

## 📊 数据来源
| 数据类型 | 具体内容 | 来源/说明 |
|----------|----------|-----------|
| 基础地理数据 | 全国省级行政区边界矢量数据、长三角数字高程模型（DEM） | 标准地图服务系统 |
| 土地利用数据 | 2000年、2020年两期地类数据 | Globeland30地类数据集 |
| 社会经济数据 | 长三角四省/市人口数量、夜间灯光亮度数据 | 统计年鉴、遥感反演产品 |
| 驱动因子数据 | 人口密度、距城市中心距离、平均高程、平均坡度 | 空间化处理后生成 |

## 🚀 核心研究内容与方法
### 1. 核心研究方法
- 土地利用分类方法：对研究区土地利用类型进行可视化表达
- 空间分析方法：运用ArcGIS Pro的叠加分析、缓冲区分析等工具，分析土地利用空间格局变化
- 转移矩阵分析法：构建土地利用转移矩阵，量化不同地类间转换面积与概率，明确建设用地主要来源
- 地理探测器模型法：通过因子探测、交互探测，识别建设用地扩张驱动因素及因子间交互强度

### 2. 技术路线
1. 数据预处理：遥感影像裁剪、校正、配准，矢量数据投影统一及属性完善
2. 土地利用变化分析：通过转移矩阵、面积变化率、空间格局指数等分析时空演变特征
3. 驱动因子选取与标准化：筛选5类核心驱动因子，完成数据标准化与空间化表达
4. 驱动性分析：运用地理探测器模型开展因子探测与交互探测，量化驱动因子影响力

### 3. 关键实验步骤
1. 项目创建：在ArcGIS Pro中创建“长三角地区土地利用变化及驱动性分析”项目
2. 研究区范围提取：筛选四省（市）边界要素，导出省级边界并提取市县二级行政区
3. 土地利用数据提取：通过掩膜提取Globeland30地类数据，得到2期地类数据
4. 变化数据计算：运用栅格计算器叠加两期数据，构建土地利用转移矩阵，计算面积变化量与动态度
5. 建设用地处理：重分类提取建设用地，转矢量后与县界相交汇总，计算建设用地变化率
6. 驱动因子处理：对5类驱动因子进行分区统计，连接至属性表完成数据整合
7. 地理探测器运算：导入标准化数据，开展因子探测与交互探测，量化解释力与交互作用

## 📈 核心研究结果
### 1. 土地利用总体变化特征
- 建设用地增幅最大：增加22242.66km²，变化率4.358%，核心来源为耕地转换（24110.79km²）
- 耕地大幅缩减：由197434.85km²减至175623.94km²，减幅11.17%，年均动态度-0.552%
- 空间格局：建设用地呈现“中心集聚、廊道延伸”特征，核心城市圈连片扩张，交通干线沿线形成扩张带

### 2. 建设用地空间差异
- 高值区：上海-苏州-无锡、南京-镇江、杭州-嘉兴-绍兴等核心城市圈，呈连片扩张态势
- 低值区：皖北、苏北等外围区域，以县城及重点镇为中心点状扩张，区域连通性较弱

### 3. 驱动性分析结果
- 单因子解释力排序：平均坡度（q=0.1658）>平均高程（q=0.1317）>人口密度（q=0.0645）>灯光指数（q=0.041）>距市驻地距离（q=0.03）
- 交互作用：所有驱动因子间均为“双因子增强”或“非线性增强”，距市驻地距离与平均坡度的非线性协同效应最显著（交互q=0.2488）
